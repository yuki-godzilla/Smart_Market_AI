# 銘柄ランキング画面 並べ替え情報設計整理レポート

作成日: 2026-06-01

## 1. 再整理方針

- 上部UIは `評価方針` を主導線にする。
- 単一指標ソートは上部ドロップダウンに置かず、詳細テーブルの列ソートに吸収する。
- 上位候補カード、Top 10グラフ、確認メモは、選択中の複合評価方針に基づいて表示する。

## 2. 評価方針

上部UIの候補は、SMAIの複合評価プロファイルとして以下を表示する。

- `AI総合` -> `総合マルチファクター`
- `上昇気配重視`
- `モメンタム・トレンド`
- `成長クオリティ`
- `割安クオリティ`
- `高配当の持続性`
- `低ボラ・安定`
- `リスク調整パフォーマンス`
- `小型・成長探索`
- `NISA長期適合`
- `データ信頼度優先`
- `ETF低コスト・コア`
- `ETFインカム・分散`

旧来の `配当重視` / `成長重視` / `割安重視` / `安定重視` / `トレンド重視` は、上部UIでは重複表示せず、内部互換として代表プロファイルへ寄せる。

## 3. 単一指標ソート

上部UIの単一指標 `並び替え` ドロップダウンは削除した。

`総合スコア` / `配当利回り` / `PER` / `PBR` / `ROE` / `時価総額` / `出来高` / `ボラティリティ` / `リスク` / `データ品質` / `スクリーニング` / `上昇気配` / `下降警戒` は、詳細テーブルの列名クリックで並べ替える。

## 4. 詳細テーブル

詳細テーブルでは、列名クリックによるローカルソートを維持する。

- `総合スコア`, `配当利回り`, `ROE`, `時価総額`, `出来高`, `データ品質`: 高い順
- `スクリーニング`, `上昇気配`: 高い順
- `PER`, `PBR`, `ボラティリティ`, `リスク`, `下降警戒`: 低い順
- 欠損値: `N/A` 表示で末尾

表示ガイドは `詳細テーブルでは、列名をクリックして各指標順に並べ替えできます。` から始まる文言にした。

## 5. リスク列の扱い

上部の評価方針には `低リスク候補順` のような単一指標ソート名を出さない。

詳細テーブルでは `リスク` 列を補助確認の列として扱い、列クリック時は低い順から確認できるようにする。安全保証や売買推奨ではないことはランキング全体の注意書きで維持する。
